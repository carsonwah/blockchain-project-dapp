# blockchain-project-dapp

## Getting Started

### Install Dependencies

- Node.js
- Truffle Framework
  - `npm install -g truffle`
- Ganache
  - https://truffleframework.com/ganache
- MetaMask on Google Chrome
- `npm install`

Basic structure is based on https://truffleframework.com/boxes/pet-shop.

### Useful Commands

```bash
truffle develop
truffle compile
truffle migrate
npm run dev
```
